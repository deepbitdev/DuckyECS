# DuckyECS
Created by Dre Smith

## About
DuckyECS is a data-oriented, archetype-based entity component system (ECS) for Swift.

## Features
- Fast serialization and deserialization using `Codable`.
- Efficient archetype storage and querying.
- Automatic storage compaction and defragmentation.
- Component mutation notifications.
- Optional multi-threaded systems.
